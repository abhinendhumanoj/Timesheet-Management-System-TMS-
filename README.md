# Timesheet Management System

## Overview
A role-based web application to manage employee timesheets.

## Features
- Role-based login (Admin, Manager, Resource)
- Timesheet tracking
- Approval workflow
- Reports

## Tech Stack
- Node.js
- Express.js
- EJS
- CSS

## How to Run
1. npm install
2. node app.js
3. Open http://localhost:3000
